Je werkplek organiseren

Bij workshops van de Carpentries leer je door "mee te coderen" met de docenten. Om zo te kunnen werken, moet je een Zoom software geopend hebben, én de software tool die je op dat moment leert. We raden je daarom aan om je werkplek te organiseren volgens één van de volgende opties:

  • Twee displays: Als je twee displays tot je beschikking hebt, probeer dan Zoom op één display te openen, en de software tool op de andere.
  • Twee apparaten: Als je geen twee displays hebt, kun je misschien twee apparaten gebruiken. Het is handiger om het kleinste scherm (telefoon, tablet) te gebruiken voor Zoom, en een groter scherm voor de software tool.
  • Gedeeld scherm: Als je maar één apparaat met één scherm tot je beschikking hebt, bekijk dan vooraf hoe je je scherm het handigst kan indelen. Op grotere displays kan je twee applicaties naast elkaar zetten. Op kleinere schermen werkt het misschien beter om te wisselen tussen de applicaties.
Deze blog post bespreekt verschillende mogelijkheden in detail.
